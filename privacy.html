A quick note on privacy.

No JavaScript/Cookies/Storage.

This web space is entirely JavaScript free! Feel free to disable here it anyways (always a good practice). I don't set any cookies or use the Storage API (localStorage, etc). Again - feel free to block all this.

Anything is self hosted.

All my content is self hosted. Yes, even fonts (or: especially fonts)! This prevents all kinds of trackers, like fingerprinting, canvas, and many more. Seriously, this should be a standard on any web space that values (or claims to value) the users privacy. Using third-party APIs for content is a big no-no.
P.S: Seriously, stop using Googles API just for fonts (or anything, really)..
FYI: The server is located in New Jersey (NY, United States).

What is being logged/monitored?

In order to "protect" this web space I use Cloudflare's DNS and log anything as shown below:

  • your IP address
  • request/response headers
  • bandwidth
  • the geolocation of your IP address
This data will get stored on Cloudflare's servers as well as mine.
If you feel uncomfortable and don't want to leak your IP address, then use a VPN. This won't hide your bandwidth and headers though (for obvious reasons).
