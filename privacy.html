A quick note on privacy.

No JavaScript/Cookies/localStorage.

This web space is entirely JavaScript free! Feel free to disable here it anyways (always a good practice). Neither do I set/make use of any cookies whatsoever nor use your browsers localStorage (which would depend on JavaScript anyways).

Anything is self hosted.

All my content is self hosted. Yes, even fonts (or: especially fonts)! This prevents all kinds of trackers, like fingerprinting, canvas, and many more. Seriously, this should be a standard on any web space that values (or claims to value) the users privacy. Using third-party APIs for content is a big no-no.
The server is located in New Jersey (NY, United States).

What is being logged/monitored?

In order to "protect" this web space I use log anything as shown below:

  • your IP address
  • request/response headers
  • bandwidth
This data will get stored on my server for a brief period (approx. 24-72hrs). I don't care about you or your data enough to waste storage on it.


If you feel uncomfortable and don't want to leak your IP address, then use a VPN. This won't hide your bandwidth and headers though (for obvious reasons).


By the way, the source code of this web space can be found over at my repositories!

If you want to see which changes have been made to this page (and anything really), head over to my repositories and browse the commits.
